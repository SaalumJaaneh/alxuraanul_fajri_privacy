Quraanul Fajri

Politique de confidentialité

Introduction

Quraanul Fajri est une application de lecture et d'écoute du Saint Coran. Nous prenons la protection de vos données personnelles très au sérieux. Cette politique de confidentialité explique quelles données sont collectées et comment elles sont utilisées.

Données collectées

Données de localisation

L'application peut accéder à votre position géographique uniquement pour :

  • Calculer les horaires de prière selon votre emplacement
  • Déterminer la direction de la Qibla (Kaaba)

Votre position n'est jamais envoyée à nos serveurs ni partagée avec des tiers. Elle est utilisée localement sur votre appareil et transmise uniquement à l'API Aladhan pour le calcul des horaires de prière.

Données stockées localement

Les données suivantes sont stockées exclusivement sur votre appareil :

  • Vos favoris et marque-pages
  • Votre progression de lecture
  • Vos paramètres (thème, réciteur, langue)
  • Le cache des sourates pour le mode hors-ligne

Aucune de ces données n'est envoyée à un serveur externe.

Publicités

L'application utilise Google AdMob pour afficher des bannières publicitaires. Google AdMob peut collecter certaines informations telles que l'identifiant publicitaire de votre appareil, conformément à la politique de confidentialité de Google.

Vous pouvez réinitialiser votre identifiant publicitaire ou désactiver la personnalisation des annonces dans les paramètres de votre appareil Android (Paramètres > Google > Annonces).

Services tiers

L'application communique avec les services suivants :

  • Al Quran Cloud API (api.alquran.cloud) — texte du Coran et traductions
  • EveryAyah (everyayah.com) — fichiers audio des récitations
  • Aladhan API (api.aladhan.com) — horaires de prière
  • Google AdMob — affichage de publicités

Ces services peuvent collecter des données techniques (adresse IP, type d'appareil) conformément à leurs propres politiques de confidentialité.

Sécurité

Toutes les communications avec les API externes sont effectuées via HTTPS. Les données stockées localement sur votre appareil sont protégées par les mécanismes de sécurité du système d'exploitation Android/iOS.

Enfants

Cette application est destinée à un public général. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans.

Modifications

Cette politique de confidentialité peut être mise à jour. En cas de modification, la nouvelle version sera publiée sur cette page avec la date de mise à jour.
